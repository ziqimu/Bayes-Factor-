# Bayes-Factor-Implementation
We utilize the Polya tree method to implement a nonparametric Bayesian test. This approach allows us to compare galaxy properties between dense regions (voids) and less dense regions (walls) identified by two void-detection algorithms, VoidFinder and VÂ², using data from the SDSS DR7 catalog and the DESI DR1 catalog.
